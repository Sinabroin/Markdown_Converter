# Skill: Markdown 정규화

## 처리 파이프라인 (순서 중요)
1. **줄 단위 후행 공백 제거** — `line.rstrip()`
2. **연속 개행 축소** — 3줄+ 빈 줄 → 2줄 (`\n\n`)
3. **불릿 통일** — `*`, `+` 시작 불릿을 `-`로 교체 (코드블록 내부 제외)
4. **빈 헤딩 제거** — `## ` 뒤에 내용 없는 헤딩 라인 삭제
5. **반복 토큰 축소** — 동일 10자+ 문자열이 연속 3회+ → 1회로 축소
6. **선행/후행 빈 줄 제거** — 문서 시작/끝 빈 줄 strip

## 코드블록 보호
정규화는 코드블록(`` ``` ``으로 감싸진 영역) 내부를 건드리지 않는다.
구현: 코드블록을 플레이스홀더로 치환 → 정규화 → 복원

## 하지 않는 것
- 내용 생성·수정 (의미 변경 금지)
- URL 날조 (링크 라벨만 정리, 타겟 URL은 원본 유지)
- 헤딩 레벨 자동 재배치 (원본 구조 존중)

## 독립 실행
```powershell
python scripts\normalize_md.py output\raw_md\sample.md
# → output\clean_md\sample.md 에 저장
```
